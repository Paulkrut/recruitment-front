# 🎬 Модуль воспроизведения процесса печати

## Описание

Модуль позволяет HR-менеджеру увидеть **как кандидат печатал** в реальном времени, включая:
- ⏱️ Скорость печати в каждый момент
- ⏸️ Паузы (где кандидат задумывался)
- ⌫ Исправления (Backspace)
- 📊 График прогресса

---

## Архитектура

### **1. Компонент `TypingReplayPlayer.tsx`** (Отдельный модуль)

**Путь:** `recruitment-front/src/components/hr/TypingReplayPlayer.tsx`

**Функционал:**
- Воспроизведение процесса печати с учётом реальных временных интервалов
- Регулировка скорости (1x, 2x, 4x, 8x)
- Контролы: Play, Pause, Stop
- Подсветка исправлений (Backspace)
- Отображение текущей скорости печати
- Прогресс-бар

**Входные данные:**
```typescript
interface TimelineEvent {
  timestamp: number;      // Unix timestamp в миллисекундах
  char: string;           // Символ ('a', 'b', '' для Backspace)
  action: 'add' | 'delete'; // Добавление или удаление
}
```

**Пример:**
```json
[
  { "timestamp": 1706291234567, "char": "П", "action": "add" },
  { "timestamp": 1706291234789, "char": "р", "action": "add" },
  { "timestamp": 1706291235456, "char": "", "action": "delete" },
  { "timestamp": 1706291235789, "char": "р", "action": "add" }
]
```

## Технические детали

### **Алгоритм воспроизведения:**

1. Берём первое событие из `timeline`
2. Вычисляем задержку до следующего события: `delay = nextEvent.timestamp - currentEvent.timestamp`
3. Применяем ускорение: `delay = delay / speed`
4. Ограничиваем максимальную задержку: `delay = Math.min(delay, 500)` (даже длинные паузы не превышают 500мс)
5. Обновляем текст:
   - `action === 'add'` → добавляем символ
   - `action === 'delete'` → удаляем последний символ
6. Показываем анимированный курсор `|`
7. Обновляем метрики (текущая скорость, прогресс)
8. Планируем следующее событие через `setTimeout`

### **Оптимизации:**

- ✅ Максимальная задержка ограничена 500мс (для ускорения длинных пауз)
- ✅ Cleanup через `useEffect` при размонтировании
- ✅ Возможность паузы/остановки в любой момент
- ✅ Регулировка скорости без перезапуска

### **UX улучшения:**

- 🎨 Анимация курсора (мигающий `|`)
- 🔴 Подсветка исправлений (Backspace) красным чипом
- 📊 Показ текущей скорости печати
- 🎚️ Прогресс-бар с цветовой индикацией (синий = играет, оранжевый = пауза, зелёный = завершено)

## Польза для HR

### **1. Выявление неуверенности:**
```
Быстрая печать: "Я работал в компании" ⚡
Пауза 5 секунд... 🟡
Медленная печать: "разработчиком Python" 🐌
```
→ **Вывод:** Кандидат не уверен в формулировке про Python

### **2. Обнаружение "придумывания":**
```
Нормальная скорость: "Мой опыт составляет"
Много Backspace: "3 года" ⌫⌫⌫ "5 лет" ⌫⌫⌫ "4 года"
```
→ **Вывод:** Кандидат колеблется с опытом (возможно завышает)

### **3. Оценка навыков письма:**
```
Быстрая и чистая печать (мало Backspace) ⚡✅
```
→ **Вывод:** Кандидат уверен, хорошие навыки письменной коммуникации

## Безопасность и приватность

- 🔒 **Конфиденциальные данные** — доступны только HR с правами
- 🚫 **НЕ передаются кандидату** (только метрики без timeline)
