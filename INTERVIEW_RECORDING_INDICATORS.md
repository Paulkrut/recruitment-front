# Визуальный индикатор записи в интервью

## 🎬 Обзор

Добавлен яркий и информативный визуальный индикатор, который делает очевидным факт ведения записи во время интервью.

---

## ✨ Индикатор записи:

### **Текстовый индикатор над кнопками управления**

```typescript
{recording && (
  <Box sx={{
    position: 'absolute', // НЕ в потоке документа
    top: isMobile ? -70 : -80, // Выше кнопок
    left: isMobile ? 8 : 16,
    right: isMobile ? 8 : 16,
    display: 'flex',
    alignItems: 'center',
    justifyContent: 'center',
    gap: 1.5,
    p: 2,
    bgcolor: '#fff3f3',
    borderRadius: '12px',
    border: '2px solid #ff4444',
    boxShadow: '0 2px 8px rgba(255, 68, 68, 0.2)',
    zIndex: 5
  }}>
    <Box sx={{
      width: 12,
      height: 12,
      borderRadius: '50%',
      bgcolor: '#ff4444',
      animation: `${pulse} 1s ease-in-out infinite`
    }} />
    <Typography sx={{
      fontSize: isMobile ? '14px' : '16px',
      fontWeight: 'bold',
      color: '#ff4444',
      letterSpacing: '0.5px'
    }}>
      Идёт запись... Говорите в {cameraEnabled ? 'камеру' : 'микрофон'}
    </Typography>
    {/* Анимированные полоски (эквалайзер) */}
    <Box sx={{ display: 'flex', gap: 0.5 }}>
      {[0, 1, 2].map((i) => (
        <Box
          key={i}
          sx={{
            width: 4,
            height: 16,
            bgcolor: '#ff4444',
            borderRadius: '2px',
            animation: `${pulse} 1s ease-in-out infinite`,
            animationDelay: `${i * 0.15}s`
          }}
        />
      ))}
    </Box>
  </Box>
)}
```

**Эффект:**
- Большой красный блок с текстом
- **Абсолютное позиционирование** - не увеличивает высоту блока кнопок ✅
- Расположен поверх области чата (не сжимает её) ✅
- Пульсирующая точка слева
- Текст адаптируется: "камеру" или "микрофон"
- Анимированные полоски справа (эквалайзер-эффект)
- Адаптивный отступ сверху для mobile/desktop

---

## 🎨 Используемая анимация:

### **`pulse` (пульсация)**
```typescript
const pulse = keyframes`
  0% { opacity: 1 }
  50% { opacity: 0.5 }
  100% { opacity: 1 }
`;
```

Плавная пульсация от полной прозрачности к полупрозрачности и обратно.

Используется для:
- Пульсирующей красной точки слева
- Анимированных полосок эквалайзера справа (с задержкой)

## 🔧 Технические детали:

### Условие показа:
Индикатор отображается только когда `recording === true`

### Производительность:
- CSS анимации (не JavaScript) - аппаратное ускорение
- Анимации используют `opacity` - оптимизированное свойство
- `will-change` не требуется (простые анимации)

### z-index:
- Текстовый индикатор: `zIndex: 5` (поверх области чата)
- Остальные элементы в нормальном потоке

### Абсолютное позиционирование:
Текстовый индикатор использует `position: absolute` чтобы:
- ✅ **Не увеличивать высоту блока кнопок**
- ✅ **Не сжимать область чата**
- ✅ **Плавать поверх интерфейса**

Отступ сверху адаптивный:
- Desktop: `-80px` (выше на 80px от кнопок)
- Mobile: `-70px` (выше на 70px от кнопок)
