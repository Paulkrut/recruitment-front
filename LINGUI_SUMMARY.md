## 🚀 Как использовать

### Базовое использование

```tsx
import { Trans, useLingui } from '@lingui/react';
import { msg } from '@lingui/macro';

function MyComponent() {
  const { _ } = useLingui();
  
  return (
    <div>
      {/* Текст */}
      <h1><Trans>Заголовок</Trans></h1>
      
      {/* Атрибуты */}
      <input placeholder={_(msg`Введите текст`)} />
      
      {/* С переменными */}
      <p><Trans>Привет, {userName}!</Trans></p>
    </div>
  );
}
```

### Workflow

```bash
# 1. Напишите код с <Trans>
# 2. Извлеките тексты
npm run i18n:extract

# 3. Добавьте переводы в src/locales/en/messages.po
# 4. Скомпилируйте
npm run i18n:compile

# 5. Проверьте результат
npm run dev
```

## 🔧 Техническая информация

### Формат файлов

Используется **PO формат** (Portable Object):

```po
msgid "Создать вакансию"
msgstr "Create Vacancy"

msgid "Привет, {userName}!"
msgstr "Hello, {userName}!"
```

### Компиляция

`.po` файлы компилируются в оптимизированный JS:

```
messages.po → messages.js (минифицированный, tree-shakeable)
```

### Производительность

- Переводы загружаются только для текущего языка
- Компиляция происходит заранее, не в runtime
- Минимальный размер бандла
- Tree shaking для неиспользуемых переводов
