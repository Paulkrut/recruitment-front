## 🎯 Проблема и решение

**Проблема**: На мобильных устройствах появлялся горизонтальный скролл из-за широкого блока с кнопками.

**Решение**: Три разных варианта отображения в зависимости от размера экрана:
1. **Mobile (< sm)**: Одна иконка-кнопка + выпадающее меню
2. **Tablet (sm - md)**: Компактный блок без текста
3. **Desktop (md+)**: Полный блок с текстом и всеми элементами

---


**Особенности**:

- Без текста "Ваш менеджер"
- Компактный padding

### 3️⃣ Desktop (> 900px) - Полный блок

## 📊 Breakpoints

```tsx
const smUp = useMediaQuery(theme.breakpoints.up('sm'));  // >= 600px
const mdUp = useMediaQuery(theme.breakpoints.up('md'));  // >= 900px
const lgUp = useMediaQuery(theme.breakpoints.up('lg'));  // >= 1200px
```

### Логика отображения:

```tsx
if (!smUp) {
  return <IconButtonWithMenu />  // Mobile: < 600px
}

if (!mdUp) {
  return <CompactBlock />        // Tablet: 600-900px
}

return <FullBlock />             // Desktop: > 900px
```
