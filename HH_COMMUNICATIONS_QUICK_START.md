## 📖 Как пользоваться

### 1️⃣ Отправка приглашения на интервью

1. Откройте карточку кандидата из HH.ru
2. Найдите кнопку **"HH"** (с иконкой письма) в верхней части страницы
3. Нажмите на кнопку
4. ✅ Приглашение отправлено!
5. Сообщение **автоматически** появится во вкладке "Коммуникации" через 1-2 секунды

### 2️⃣ Просмотр истории коммуникаций

1. В карточке кандидата перейдите на вкладку **"💬 Коммуникации"**
2. Увидите timeline со всеми сообщениями:
   - 📤 **Синяя рамка** = отправлено от нас
   - 📥 **Оранжевая рамка** = получено от кандидата
   - 👤 Имя HR-сотрудника
   - 📅 Дата и время
   - 💬 Текст сообщения

### 3️⃣ Загрузка сообщений из HH.ru вручную

1. Откройте вкладку **"💬 Коммуникации"**
2. Нажмите кнопку **"🔄 Загрузить из HH"** (справа вверху)
3. Подождите пару секунд
4. ✅ Новые сообщения загружены и отображены

## 🎯 Что происходит автоматически

- ✅ После отправки приглашения сообщения **автоматически синхронизируются**
- ✅ Дубликаты **не создаются** (проверка по ID сообщения из HH)
- ✅ При открытии вкладки "Коммуникации" данные **автоматически загружаются** (если еще не загружены)

## 🧪 Тестирование

### Сценарий 1: Отправка приглашения
1. Откройте кандидата из HH.ru
2. Нажмите кнопку "HH"
3. Проверьте:
   - ✅ Показывается "✅ Приглашение отправлено в HH.ru"
   - ✅ Через 1-2 сек показывается "✅ Загружено сообщений: N"
4. Перейдите на вкладку "Коммуникации"
5. Проверьте:
   - ✅ Отправленное сообщение отображается
   - ✅ Есть метка "🔗 Ссылка на интервью отправлена"
   - ✅ Указано ваше имя как отправителя

### Сценарий 2: Загрузка сообщений
1. Откройте вкладку "Коммуникации"
2. Нажмите "🔄 Загрузить из HH"
3. Проверьте:
   - ✅ Показывается loader "Загрузка..."
   - ✅ После загрузки показывается "✅ Загружено сообщений: N"
   - ✅ Все сообщения отображаются правильно
   - ✅ Входящие и исходящие сообщения имеют разные цвета

### Сценарий 3: Обработка ошибок
1. Отключите интеграцию HH.ru (отзовите токен)
2. Попробуйте загрузить сообщения
3. Проверьте:
   - ✅ Показывается понятная ошибка
   - ✅ Токен деактивирован в настройках

## ❓ Возможные проблемы

### "Кнопка 'Загрузить из HH' не отображается"
- У кандидата нет переписки в HH.ru
- Кнопка показывается только если `hasHhNegotiation === true`

### "Сообщения не загружаются"
- Проверьте токен HH.ru в настройках интеграции
- Проверьте что у кандидата есть `negotiation_id`
- Проверьте логи: `recruitment/var/log/dev.log`

### "Ошибка 403"
- Требуется платный тариф HH.ru с доступом к API работодателя
- Бесплатный тариф HH.ru не поддерживает API сообщений
