## 📊 User Flow

### **Сценарий 1: Заканчивается баланс**

```
Пользователь видит красный индикатор в header
↓
Кликает на баланс
↓
Открывается меню с предупреждением "Баланс заканчивается!"
↓
Кликает "Пополнить баланс"
↓
Переход на /hr/billing → выбор тарифа → оплата
```

### **Сценарий 2: Изучение тарифов**

```
Пользователь хочет изучить тарифы
↓
Открывает sidebar
↓
Видит секцию "💰 Биллинг и тарифы"
↓
Переходит в нужный раздел:
  - Тарифные планы (сравнение)
  - История операций (что уже куплено)
  - Аналитика (как использовал ресурсы)
```

### **Сценарий 3: Посмотреть аналитику**

```
Пользователь хочет понять, как расходует интервью
↓
Вариант A: Клик на баланс в header → "Аналитика"
Вариант B: Sidebar → "Биллинг и тарифы" → "Аналитика"
↓
Переход на /hr/billing/analytics
↓
Видит детальную статистику с градиентными картами
```

## 🎨 Цветовая схема (sidebar)

```tsx
bgcolor настройки для визуального различия:

Основные разделы:
  Главная            → primary   (синий)
  Вакансии           → warning   (оранжевый)
  Кандидаты          → secondary (фиолетовый)
  Брендирование      → success   (зелёный)
  Компании           → warning   (оранжевый)
  Сотрудники         → info      (голубой)
  HH.ru              → error     (красный)

Биллинг:
  Тарифные планы     → primary   (синий)
  История операций   → info      (голубой)
  Аналитика          → success   (зелёный)

Тестирование:
  Регламенты         → success   (зелёный)
  Тесты              → info      (голубой)

Система:
  Настройки          → grey      (серый)
  Запросы удаления   → warning   (оранжевый)
```
