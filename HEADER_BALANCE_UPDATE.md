## Дата: 24 октября 2025

## ✅ Что было сделано

### 1. Создан компонент **BalanceInfo.tsx**

Отображает текущий баланс интервью и тестов с подробной информацией.

**Основные возможности:**
- 💼 Баланс интервью (текущий / всего)
- 📋 Баланс тестов по регламентам (текущий / всего)
- ⚠️ Индикация низкого баланса (красный фон при < 3 интервью или < 5 тестов)
- 📊 Прогресс-бары с цветовой индикацией:
  - 🟢 Зелёный: > 50%
  - 🟡 Жёлтый: 20-50%
  - 🔴 Красный: < 20%

**Dropdown меню (при клике):**
- Детальная информация о балансе
- Статистика за всё время:
  - Проведено интервью
  - Проведено тестов
  - Потрачено денег
- Предупреждение при низком балансе
- Кнопка "Пополнить баланс" → `/hr/billing`
- Кнопка "История операций" → `/hr/billing/transactions`

**Автообновление:**
- Загружается при монтировании компонента
- API: `GET /api/billing/stats`

### 2. Создан компонент **CompanyInfo.tsx**

Отображает информацию о текущей компании.

**Что показывает:**
- 🏢 Логотип компании (или иконку по умолчанию)
- Название компании
- ИНН (если заполнен)

**Адаптивность:**
- Показывается только на экранах **lg** и выше (> 1200px)
- На мобильных скрыт для экономии места

**Автозагрузка:**
- API: `GET /api/companies/{id}`
- Использует `currentCompanyId` из localStorage

## 🎯 Цветовая индикация баланса

### В топбаре:
- **Нормально** (> 3 интервью): белый фон, обычная граница
- **Низкий баланс** (≤ 2 интервью или ≤ 5 тестов):
  - 🔴 Красный фон
  - ⚠️ Иконка предупреждения
  - Красная граница

### В прогресс-барах:
- **🟢 Зелёный** (> 50%): Всё отлично
- **🟡 Жёлтый** (20-50%): Скоро закончится
- **🔴 Красный** (< 20%): Критически мало

## 🔌 API Endpoints используемые

### 1. Получение баланса
```
GET /api/billing/stats
Headers:
  - Authorization: Bearer {token}
  - X-Company-ID: {companyId}

Response:
{
  "current_balance": {
    "interviews": 8,
    "regulation_tests": 15
  },
  "lifetime": {
    "interviews_purchased": 10,
    "interviews_used": 2,
    "regulation_tests_purchased": 20,
    "regulation_tests_used": 5,
    "total_spent": "3500.00"
  },
  "usage_rate": 20.0
}
```

### 2. Получение информации о компании
```
GET /api/companies/{id}
Headers:
  - Authorization: Bearer {token}
  - X-Company-ID: {companyId}

Response:
{
  "id": 1,
  "name": "ООО Рекрутмент",
  "inn": "1234567890",
  "logo": "/uploads/branding/company-logo.png"
}
```
