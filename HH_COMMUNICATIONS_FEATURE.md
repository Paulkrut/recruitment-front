# 💬 Коммуникации с кандидатами из HH.ru

## 📋 Описание

Добавлена полная система отслеживания коммуникаций с кандидатами из HeadHunter.ru. Теперь вся история переписки сохраняется и отображается в удобном timeline-интерфейсе.

## ✨ Что реализовано

### Backend

#### 1. **Автоматическое логирование отправки приглашений**
   - При отправке приглашения на интервью через HH.ru создается запись в `hh_communication`
   - Сохраняется:
     - Текст сообщения
     - Ссылка на интервью
     - Кто отправил (HR-сотрудник)
     - Дата и время отправки
     - ID сообщения из HH.ru (для синхронизации)

**Файлы:**
- `recruitment/src/Controller/Api/HhIntegrationController.php` - метод `sendInterviewLink()`

#### 2. **API для загрузки сообщений из HH.ru**
   - **Endpoint:** `POST /api/hh-integration/candidate/{id}/sync-messages`
   - Загружает все сообщения из переписки с кандидатом в HH.ru
   - Автоматически определяет направление (входящие/исходящие)
   - Пропускает дубликаты (проверка по `hh_message_id`)
   - Сохраняет метаданные (автор, статус прочтения и т.д.)

**Файлы:**
- `recruitment/src/Service/HhApiService.php` - метод `getNegotiationMessages()`
- `recruitment/src/Controller/Api/HhIntegrationController.php` - метод `syncMessages()`

#### 3. **API для получения списка коммуникаций**
   - **Endpoint:** `GET /api/admin/candidates/{id}/communications`
   - Возвращает все коммуникации для кандидата
   - Сортировка по дате (новые сначала)
   - Поддержка разных источников (HH, email в будущем)

**Файлы:**
- `recruitment/src/Controller/Api/AdminCandidateController.php` - метод `getCommunications()`

#### 4. **Структура базы данных**
Используется существующая таблица `hh_communication`:

```sql
CREATE TABLE hh_communication (
    id INT AUTO_INCREMENT PRIMARY KEY,
    hh_candidate_id INT NOT NULL,
    type VARCHAR(50) NOT NULL, -- interview_invitation_sent, message_sent, message_received
    direction VARCHAR(20) NOT NULL, -- outgoing, incoming, internal
    message TEXT,
    metadata JSON, -- доп. данные (ссылки, файлы и т.д.)
    sent_by INT, -- кто отправил (User)
    created_at DATETIME NOT NULL,
    hh_message_id VARCHAR(255), -- ID сообщения в HH для синхронизации
    is_read BOOLEAN DEFAULT FALSE,
    FOREIGN KEY (hh_candidate_id) REFERENCES hh_candidate(id),
    FOREIGN KEY (sent_by) REFERENCES user(id)
);
```

### Frontend

#### 1. **Вкладка "Коммуникации"**
   - Переименована вкладка "Письма" → "Коммуникации"
   - Красивый timeline-интерфейс с иконками
   - Цветовая индикация направления сообщений:
     - 🔵 Синяя граница - исходящие (от нас)
     - 🟠 Оранжевая граница - входящие (от кандидата)

**Файлы:**
- `recruitment-front/src/app/(DashboardLayout)/hr/candidates/[id]/page.tsx`

#### 2. **Кнопка "Загрузить из HH"**
   - Видна только если у кандидата есть переписка в HH.ru
   - При нажатии загружает все сообщения из HH.ru
   - Показывает прогресс загрузки
   - Отображает количество загруженных сообщений

#### 3. **Автоматическая синхронизация**
   - После отправки приглашения на интервью автоматически загружаются сообщения из HH.ru
   - Задержка 1 секунда (чтобы HH.ru успел обработать)
   - Тихая загрузка в фоне

#### 4. **Отображение коммуникаций**
Каждая запись показывает:
- 💬 Иконка источника (HH.ru / email)
- 📤/📥 Направление (исходящие/входящие)
- 👤 Кто отправил (имя HR-сотрудника)
- 📅 Дата и время
- 💬 Текст сообщения
- 🔗 Метка "Ссылка на интервью отправлена" (если есть)

## 🎯 Типы коммуникаций

### Исходящие (от нас)
- `interview_invitation_sent` - Отправлено приглашение на интервью
- `message_sent` - Отправлено сообщение в HH.ru

### Входящие (от кандидата)
- `message_received` - Получено сообщение из HH.ru

### Внутренние (в будущем)
- `internal_comment` - Внутренний комментарий HR

## 📖 Инструкция для пользователя

### Как посмотреть коммуникации с кандидатом

1. Откройте карточку кандидата в HR-панели
2. Перейдите на вкладку "💬 Коммуникации"
3. Если коммуникаций нет, они загрузятся автоматически (если есть)

### Как загрузить сообщения из HH.ru

1. Откройте вкладку "💬 Коммуникации"
2. Нажмите кнопку "🔄 Загрузить из HH" (справа вверху)
3. Дождитесь завершения загрузки
4. Новые сообщения появятся в списке

### Как отправить приглашение на интервью

1. В карточке кандидата (сверху) найдите кнопку "HH" с иконкой письма
2. Нажмите на кнопку
3. Приглашение будет отправлено в HH.ru
4. Через 1-2 секунды оно автоматически появится во вкладке "Коммуникации"

## 🔧 Технические детали

### Обработка ошибок

#### HH.ru API
- **403 Forbidden** - Требуется платный тариф HH.ru с доступом к API работодателя
- **404 Not Found** - Переписка не найдена или удалена
- **429 Too Many Requests** - Превышен лимит запросов, попробуйте позже
- **token-revoked** - Токен отозван, требуется повторная авторизация (токен автоматически деактивируется)

#### Фронтенд
- При ошибке загрузки показывается Alert с описанием
- При синхронизации показывается loader
- Автоматическое обновление списка после успешной синхронизации

### API Endpoints

```typescript
// Отправить приглашение на интервью
POST /api/hh-integration/candidate/{hhCandidateId}/send-interview-link
Response: { success: true, message: 'hh.message_sent' }

// Синхронизировать сообщения из HH.ru
POST /api/hh-integration/candidate/{hhCandidateId}/sync-messages
Response: { success: true, synced: 3, skipped: 1, total: 4 }

// Получить все коммуникации кандидата
GET /api/admin/candidates/{candidateId}/communications
Response: {
  communications: [
    {
      id: 1,
      type: 'interview_invitation_sent',
      direction: 'outgoing',
      message: 'Здравствуйте! Приглашаем вас...',
      source: 'hh',
      createdAt: '2025-12-07T14:23:00+00:00',
      sentBy: { id: 1, name: 'Артём' },
      metadata: { interview_link: '...' },
      isRead: false
    }
  ],
  hhCandidateId: 123,
  hasHhNegotiation: true
}
```

## 🧪 Тестирование

### Ручное тестирование

1. **Отправка приглашения:**
   - Откройте карточку кандидата из HH.ru
   - Нажмите кнопку "HH"
   - Проверьте что сообщение появилось во вкладке "Коммуникации"

2. **Загрузка сообщений из HH:**
   - Перейдите на вкладку "Коммуникации"
   - Нажмите "Загрузить из HH"
   - Проверьте что сообщения загрузились и отображаются правильно

3. **Автосинхронизация:**
   - Отправьте приглашение
   - Подождите 2-3 секунды
   - Перейдите на вкладку "Коммуникации"
   - Убедитесь что отправленное сообщение уже там

4. **Обработка ошибок:**
   - Попробуйте загрузить сообщения при отозванном токене HH.ru
   - Убедитесь что показывается понятная ошибка
   - Проверьте что токен деактивирован в настройках
