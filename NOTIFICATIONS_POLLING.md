# 🔄 Polling уведомлений - Real-time обновления

## ✨ Что реализовано

Автоматическое обновление счётчика уведомлений каждые **2 минуты** с визуальным feedback.

---

## 🎯 Как работает

### 1. Автоматическая проверка
```typescript
// Каждые 2 минуты (120 секунд)
setInterval(() => {
  loadUnreadCount();
}, 120000);
```

### 2. Лёгкий запрос
```typescript
// Запрашиваем только счётчик, без загрузки списка
GET /api/notifications?unread_only=true&limit=1

Response: { 
  success: true, 
  unread_count: 5 
}
```

### 3. Визуальный feedback

**При появлении новых уведомлений:**
- ✅ 🔔 Иконка: **пульсирует** 3 раза (увеличение/уменьшение)
- ✅ 🔴 Бейдж: **светится** красным 3 раза
- ✅ Счётчик обновляется автоматически
- ✅ Анимация длится 3 секунды

## 📊 Производительность

**Что запрашивается:**
- ✅ Только счётчик (1 число)
- ✅ Без загрузки уведомлений
- ✅ Легкий запрос (~50ms)

**Частота:**
- ⏱️ Каждые 2 минуты
- 📊 30 запросов/час на пользователя
- 💾 Минимальная нагрузка на сервер
