# Функционал "Переписать/Отправить" - Краткая инструкция

## 🎯 Что изменилось?

Теперь при прохождении интервью кандидаты могут **переписывать** свои ответы перед отправкой!

## 🔄 Как это работает?

### 3 состояния интерфейса:

#### 1️⃣ **Готов к записи**
```
┌─────────────────────┐  ┌──────────────┐
│ 🎤 Записать ответ  │  │ ⏭️ Пропустить │
└─────────────────────┘  └──────────────┘
```

#### 2️⃣ **Идёт запись** (индикатор REC мигает)
```
┌───────────────────────┐
│ ⏹️ Остановить запись  │
└───────────────────────┘
```

#### 3️⃣ **Запись остановлена** (НОВОЕ!)
```
┌────────────────┐  ┌──────────────────────┐
│ 🔄 Переписать  │  │ ✓ Отправить ответ   │
└────────────────┘  └──────────────────────┘
```

## ✅ Особенности

- ✅ **Бесконечное количество переписываний** (пока не истёк таймер)
- ✅ **Таймер НЕ останавливается** при остановке записи
- ✅ **Работает для всех вопросов** (и видео, и аудио режим)
- ✅ **Нет preview** после остановки (для простоты)
- ✅ **Старая реплика удаляется** при перезаписи (чат остаётся чистым)
- ✨ **Красивая анимация удаления** - fade-out + slide эффект (400ms)

## 📱 Пример использования

```typescript
// 1. Кандидат записывает ответ
startRecording() 
  → Идёт запись → REC 🔴
  → В чате появляется реплика "🎤 Запись аудио..." или видео

// 2. Останавливает запись
stopRecording()
  → Blob сохранён в recordedBlob
  → Реплика в чате обновляется:
     • Видео: live preview → готовое видео для просмотра
     • Текст: "🎥 Запись готова. Выберите действие ниже"
     • Аудио: "🎤 Запись готова. Выберите действие ниже"
  → Показываются 2 кнопки

// 3. Кандидат выбирает:
// Вариант A: Не понравился ответ
handleRetake()
  → recordedBlob очищается
  → setDeletingMessageIndex(index) // Запуск анимации
  → Старая реплика плавно исчезает с fade-out + slide (400ms) ✨
  → УДАЛЯЕТСЯ из чата после завершения анимации
  → Запись начинается заново
  → Появляется новая реплика "🎤 Запись аудио..."

// Вариант B: Ответ устраивает
handleSubmitRecording()
  → sendBlobAnswer(recordedBlob)
  → Отправка на сервер
  → Реплика обновляется: "🎤 Аудио ответ отправлен"
  → recordedBlob очищается
```

## 🧪 Быстрый тест

1. Откройте интервью: `http://localhost:3002/interview/[token]`
2. Нажмите "Согласен" и "Начать интервью"
3. Нажмите "🎤 Записать ответ"
4. Скажите что-нибудь, затем "⏹️ Остановить запись"
5. Увидите 2 кнопки: "🔄 Переписать" и "✓ Отправить ответ"
6. Попробуйте нажать "🔄 Переписать" - запись начнётся заново
7. Остановите снова и нажмите "✓ Отправить ответ"

## 🐛 Потенциальные проблемы и решения

### Проблема: Кнопки не появляются после остановки записи
**Решение:** Проверьте консоль браузера, возможно ошибка в `mediaRecorder.stop()`

### Проблема: Таймер истёк, а я не успел переписать
**Ответ:** Это ожидаемое поведение - таймер НЕ останавливается. Успевайте за отведённое время!

### Проблема: После "Переписать" видео/аудио не работает
**Решение:** Проверьте права доступа к камере/микрофону в браузере
