# ✅ Навигация по папкам при создании теста регламентов - ГОТОВО!

## 📝 Что реализовано

✅ **Отображение папок с регламентами**
- Папки видны на корневом уровне
- Показывается количество регламентов в каждой папке
- Визуальное отличие от регламентов (большая иконка 📁)

✅ **Переход в папки**
- Клик по папке открывает её содержимое
- Внутри папки видны только регламенты этой папки
- Можно выбирать регламенты (checkbox)

✅ **Возврат к корню**
- Кнопка "⬅ Назад к корню" для выхода из папки
- Breadcrumbs показывает текущее местоположение
- Выбранные регламенты сохраняются при навигации

✅ **Выбор из разных папок**
- Можно зайти в папку, выбрать регламенты
- Вернуться назад, зайти в другую папку
- Выбрать ещё регламенты
- Все выборы сохраняются

✅ **Счётчик регламентов**
- На каждой папке показано количество регламентов
- В итоге показано общее количество выбранных регламентов
- Показано итоговое количество вопросов

✅ **Breadcrumbs навигация**
- Показывает где вы находитесь
- Формат: "/ Название папки"
- Кнопка "Назад к корню" для быстрого возврата

## 🧪 Как тестировать

### Сценарий 1: Базовая навигация
1. Перейти на `/hr/regulation-tests/create`
2. Заполнить название теста
3. Кликнуть "Далее"
4. На шаге "Выбор регламентов":
   - ✅ Должны быть видны папки с иконками 📁
   - ✅ Должно быть видно количество регламентов в каждой папке
   - ✅ Должны быть видны регламенты без папки (если есть)

### Сценарий 2: Переход в папку
1. Кликнуть на папку "HR документы"
2. ✅ Должны увидеть:
   - Кнопку "⬅ Назад к корню"
   - Breadcrumb "/ HR документы"
   - Список регламентов этой папки
3. Выбрать 2 регламента (checkbox)
4. ✅ Внизу должно быть: "Выбрано регламентов: 2"

### Сценарий 3: Выбор из разных папок
1. Зайти в папку "HR документы"
2. Выбрать 2 регламента
3. Кликнуть "Назад к корню"
4. Зайти в папку "Технические инструкции"
5. Выбрать 1 регламент
6. ✅ Должно быть: "Выбрано регламентов: 3"
7. ✅ При создании теста должны использоваться все 3 регламента

### Сценарий 4: Пустая папка
1. Создать папку без регламентов
2. Зайти в эту папку
3. ✅ Должно отобразиться: "В этой папке пока нет регламентов"

### Сценарий 5: Предвыбранный регламент
1. Перейти по ссылке `/hr/regulation-tests/create?regulationId=5`
2. ✅ Регламент должен быть автоматически выбран
3. ✅ Должно отобразиться зелёное уведомление
4. Можно зайти в папки и выбрать дополнительные регламенты

## 💡 Примеры использования

### Кейс 1: Тест для новых сотрудников
HR создаёт тест для онбординга:
1. Заходит в папку "HR документы"
2. Выбирает "Регламент адаптации"
3. Возвращается назад
4. Заходит в папку "Безопасность"
5. Выбирает "Техника безопасности"
6. Создаёт тест на 10 вопросов (по 5 на каждый регламент)

### Кейс 2: Комплексный тест
HR создаёт большой тест:
1. Выбирает 2 регламента из "HR"
2. Выбирает 3 регламента из "Продажи"
3. Выбирает 1 регламент из "Финансы"
4. Создаёт тест на 30 вопросов (по 5 на каждый)

### Кейс 3: Быстрое создание
HR кликает "Создать тест" в списке регламентов:
1. Регламент уже выбран автоматически
2. Может зайти в папки и добавить ещё
3. Быстро создаёт тест
