# 🚀 Быстрый старт: Навигация по папкам при создании теста

## ✅ Что реализовано

При создании теста по регламентам теперь можно выбирать регламенты из разных папок.

---

## 📝 Как использовать

### 1. Создайте тест
```
/hr/regulation-tests/create
```

### 2. На шаге "Выбор регламентов"

#### В корне видны:
- **📁 Папки** - кликните чтобы зайти
- **☐ Регламенты без папки** - можно сразу выбрать

#### Внутри папки:
- **⬅ Назад к корню** - вернуться
- **☐ Регламенты** - выберите нужные

#### Выбор из разных папок:
1. Зайдите в папку "HR документы"
2. Выберите 2 регламента
3. Вернитесь назад
4. Зайдите в "Технические инструкции"
5. Выберите ещё 1 регламент
6. ✅ Выбрано: 3 регламента

---

## 📊 Визуальные подсказки

| Элемент | Вид | Действие |
|---------|-----|----------|
| Папка | 📁 Большая иконка | Клик = зайти в папку |
| Регламент | ☐ Checkbox | Клик = выбрать/отменить |
| Выбранный | ✅ Голубой фон | Регламент добавлен в тест |
| Breadcrumb | `⬅ Назад / Папка` | Показывает где вы сейчас |

## 🔧 Технические детали

### Файлы
- `recruitment-front/src/app/(DashboardLayout)/hr/regulation-tests/create/page.tsx`

### API
- `GET /api/regulations/folders` - список папок
- `GET /api/regulations` - список регламентов (с `folderId`)

### Состояние
- `selectedFolderId` - текущая открытая папка (null = корень)
- `selectedRegulations` - массив ID выбранных регламентов
- `displayedRegulations` - фильтрованный список для отображения
