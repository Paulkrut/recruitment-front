# Экспорт данных кандидата в PDF

## Обзор

Добавлена функциональность экспорта информации о кандидате в PDF формат с поддержкой кириллических шрифтов.

## Функциональность

### Что экспортируется

1. **Основная информация о кандидате:**
   - Имя кандидата
   - Email
   - Телефон
   - Статус
   - Дата создания
   - Дата завершения

2. **Детали интервью:**
   - Статус сессии
   - Время начала и завершения
   - Количество ответов
   - Общая оценка

3. **Ответы на вопросы:**
   - Текст вопроса
   - Ответ кандидата
   - Оценка за ответ
   - AI-характеристика (если есть)
   - Время ответа

4. **AI-оценка:**
   - Статус оценки
   - Резюме
   - Сильные стороны
   - Слабые стороны
   - Метрики оценки

### Особенности

- ✅ Поддержка кириллических шрифтов
- ✅ Автоматический выбор рабочего шрифта
- ✅ Автоматический перенос строк
- ✅ Разбивка на страницы
- ✅ Структурированный формат
- ✅ Перевод метрик на русский язык
- ✅ Обработка ошибок

## Использование

### На странице кандидата

1. Откройте страницу кандидата: `/hr/candidates/[id]`
2. Найдите кнопку "PDF" рядом с кнопкой "Интервью"
3. Нажмите на кнопку для экспорта
4. PDF файл автоматически скачается с именем: `candidate_[имя]_[дата].pdf`

### Программное использование

```typescript
import { exportCandidateToPDFUniversal } from '@/utils/pdfExportUniversal';

const pdfData = {
  candidate: 'Иван Иванов',
  email: 'ivan@example.com',
  phone: '+7 999 123-45-67',
  status: 'active',
  createdAt: '2024-01-01',
  finishedAt: '2024-01-02',
  sessionDetail: { /* данные сессии */ },
  evalData: { /* данные AI-оценки */ }
};

await exportCandidateToPDFUniversal(pdfData);
```

## Технические детали

### Зависимости

- `jspdf` - основная библиотека для создания PDF
- `jspdf-font` - поддержка шрифтов

### Файлы

- `src/utils/pdfExportUniversal.ts` - универсальная версия с автоопределением шрифта
- `src/utils/pdfExportCyrillic.ts` - версия с шрифтом times
- `src/utils/pdfExportCourier.ts` - версия с шрифтом courier
- `src/utils/pdfExportSymbol.ts` - версия с шрифтом symbol
- `src/utils/pdfExport.ts` - базовая версия
- `src/utils/pdfExportAdvanced.ts` - продвинутая версия

### Шрифты

Система автоматически тестирует и выбирает рабочий шрифт из списка:
1. `helvetica` - стандартный шрифт jsPDF
2. `times` - шрифт Times New Roman
3. `courier` - моноширинный шрифт
4. `symbol` - символьный шрифт

Если ни один шрифт не работает, используется `helvetica` как fallback.

### Форматирование

- Размер страницы: A4
- Ориентация: Портретная
- Единицы измерения: миллиметры
- Отступы: 20mm
- Размеры шрифтов: 10-24pt

## Обработка ошибок

Функция включает обработку ошибок:

```typescript
try {
  await exportCandidateToPDFUniversal(pdfData);
} catch (error) {
  console.error('Ошибка при экспорте PDF:', error);
  // Показать уведомление пользователю
}
```

## Ограничения

1. **Размер файла:** PDF создается на клиенте, размер зависит от количества данных
2. **Шрифты:** Используются только встроенные шрифты jsPDF
3. **Изображения:** Аудио/видео файлы не включаются в PDF (только ссылки)
4. **Стили:** Минимальное форматирование, основное внимание на читаемость

## Альтернативные версии

Если основная версия не работает, можно использовать альтернативные:

```typescript
// Версия с шрифтом times
import { exportCandidateToPDFCyrillic } from '@/utils/pdfExportCyrillic';

// Версия с шрифтом courier
import { exportCandidateToPDFCourier } from '@/utils/pdfExportCourier';

// Версия с шрифтом symbol
import { exportCandidateToPDFSymbol } from '@/utils/pdfExportSymbol';
```
