## 🎯 Почему изменили

### Преимущества новых названий:

1. **Международная терминология** - термины Hard Skills и Soft Skills понятны везде
2. **Четкое разделение** - сразу ясно, что это разные типы навыков
3. **Профессиональный стандарт** - так принято в HR-индустрии
4. **Меньше путаницы** - "знания" и "компетенции" звучали слишком похоже

### Что оценивается:

| 🎓 Hard Skills | 💡 Soft Skills |
|---------------|---------------|
| Профессиональные знания | Мотивация к вакансии |
| Технические навыки | Культура речи |
| Опыт работы | Клиентоориентированность |
| Правильность ответов | Стрессоустойчивость |
| Специализированные умения | Ответственность |
| - | Системное мышление |
