# ТЗ: Бесплатные HR-инструменты

## 📋 Философия проекта

### Принцип: "Value-First" (Ценность прежде всего)

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🎁 Бесплатно. Без регистрации. Без подвоха.          │
│                                                         │
│   Мы верим, что хорошие инструменты должны быть        │
│   доступны всем. Если вам понравится — расскажите      │
│   коллегам. Если захотите больше — мы всегда рады.     │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Что это значит:
- ✅ **100% бесплатно** — без скрытых платежей
- ✅ **Без регистрации** — не требуем email
- ✅ **Полный результат** — не урезаем функционал
- ✅ **Без навязывания** — мягкие CTA, можно игнорировать
- ✅ **Реальная помощь** — инструменты решают реальные задачи HR

### Цели:
1. **Brand Awareness** — HR-ы узнают о SofiHR
2. **Word of Mouth** — Делятся с коллегами
3. **SEO трафик** — Органический приток через поиск
4. **Долгосрочное доверие** — Когда понадобится платформа, вспомнят о нас
5. **Отложенная конверсия** — Регистрация через недели/месяцы

```php
namespace App\Controller\Api;

use App\Service\HrToolsService;
use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
use Symfony\Component\HttpFoundation\JsonResponse;
use Symfony\Component\HttpFoundation\Request;
use Symfony\Component\Routing\Annotation\Route;

#[Route('/api/public/hr-tools', name: 'api_public_hr_tools_')]

{
    public function __construct(
        private HrToolsService $hrToolsService,
        private LoggerInterface $logger
    ) {}

    #[Route('/generate-questions', name: 'generate_questions', methods: ['POST'])]
    public function generateQuestions(Request $request): JsonResponse
    {
        $data = json_decode($request->getContent(), true);
        
        $jobDescription = trim($data['jobDescription'] ?? '');
        $count = min(15, max(5, (int)($data['count'] ?? 10)));
        
        if (empty($jobDescription) || mb_strlen($jobDescription) > 5000) {
            return new JsonResponse([
                'success' => false,

            ], 400);
```

```php
<?php

namespace App\Service;

use App\Service\AI\DeepInfraDeepSeekService;

class HrToolsService
{
    public function __construct(
        private DeepInfraDeepSeekService $aiService,
        private HrToolsPrompts $prompts
    ) {}

    public function generateQuestions(string $jobDescription, int $count): array
    {
        $prompt = $this->prompts->getQuestionGeneratorPrompt($jobDescription, $count);
        
        $response = $this->aiService->generate($prompt, [
            'temperature' => 0.7,
            'max_tokens' => 2000
        ]);
        
        return $this->parseJsonResponse($response);
    }

    public function generateJobDescription(string $position, string $level, ?string $additionalInfo): array
    {
        $prompt = $this->prompts->getJobDescriptionPrompt($position, $level, $additionalInfo);
        
        $response = $this->aiService->generate($prompt, [
            'temperature' => 0.7,
            'max_tokens' => 2000
        ]);
        
        return $this->parseJsonResponse($response);
    }

    public function analyzeResume(string $resumeText, ?string $jobDescription): array
    {
        if ($jobDescription) {
            $prompt = $this->prompts->getResumeMatchPrompt($resumeText, $jobDescription);
        } else {
            $prompt = $this->prompts->getResumeAnalysisPrompt($resumeText);
        }
        
        $response = $this->aiService->generate($prompt, [
            'temperature' => 0.5,
            'max_tokens' => 2000
        ]);
        
        return $this->parseJsonResponse($response);
    }

    public function generateReply(string $type, ?string $context): array
    {
        $prompt = $type === 'accept' 
            ? $this->prompts->getAcceptReplyPrompt($context)
            : $this->prompts->getRejectReplyPrompt($context);
        
        $response = $this->aiService->generate($prompt, [
            'temperature' => 0.7,
            'max_tokens' => 500
        ]);
        
        return $this->parseJsonResponse($response);
    }

    public function getSalaryGuide(string $position, string $city, string $level, ?int $experience): array
    {
        $prompt = $this->prompts->getSalaryGuidePrompt($position, $city, $level, $experience);
        
        $response = $this->aiService->generate($prompt, [
            'temperature' => 0.3, // Более детерминированный для цифр
            'max_tokens' => 1000
        ]);
        
        return $this->parseJsonResponse($response);
    }

    private function parseJsonResponse(string $response): array
    {
        // Извлечь JSON из ответа AI
        if (preg_match('/\{[\s\S]*\}/m', $response, $matches)) {
            return json_decode($matches[0], true) ?? [];
        }
        return [];
    }
}
```
