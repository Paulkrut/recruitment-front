### **1. Confirm диалог**

**Русский (источник):**
```
⚠️ Микрофон подключен, но не обнаружен звук

Возможные причины:
• Микрофон отключен физической кнопкой
• Неправильно выбран микрофон
• Микрофон не работает

Рекомендуем скажите что-нибудь для проверки.

Всё равно начать интервью?
```

```
⚠️ Microphone connected, but no sound detected

Possible reasons:
• Microphone disabled by physical button
• Wrong microphone selected
• Microphone not working

We recommend saying something to test it.

Start interview anyway?
```

## 🎯 Использование в коде

### **Trans компонент (для JSX):**
```tsx
<Trans>🎤 Микрофон не ловит звук</Trans>
```

### **msg макрос (для строк):**
```tsx
const message = _(msg`⚠️ Микрофон подключен, но не обнаружен звук`);
confirm(message);
```

---

## 📝 Примечания

1. **Эмодзи**: Эмодзи (`🎤`, `⚠️`, `💡`, `✓`) не переводятся, но включены в строки для контекста.
2. **Многострочный текст**: В confirm диалоге используется `\n` для переносов строк.
3. **Lingui макросы**: 
   - `<Trans>` для React компонентов
   - `_(msg`...`)` для JavaScript строк
