# Улучшение проверки микрофона на странице подготовки

## 🐛 Проблема

**До исправления:**
- Система проверяла только **наличие** микрофона (`getUserMedia` успешен)
- Не проверялось, **работает ли микрофон реально**
- Кандидат мог отключить микрофон физической кнопкой на ноутбуке
- Система не замечала этого → интервью начиналось без звука

---

## ✅ Решение

### **1. Визуальный индикатор уровня микрофона**

Теперь на странице подготовки отображается:
- 🎤 **Анимированная полоса** уровня звука
- ✅ **Зелёный цвет** когда звук обнаружен
- ⚠️ **Оранжевый цвет** когда микрофон подключен, но молчит
- Текст: "Скажите что-нибудь, чтобы убедиться что микрофон работает"

### **2. Проверка перед стартом интервью**

При нажатии "Начать" если:
- ✅ Микрофон подключен (`micReady = true`)
- ❌ Уровень звука = 0 (`micLevel < 1`)

Показывается **предупреждение**:
```
⚠️ Микрофон подключен, но не обнаружен звук

Возможные причины:
• Микрофон отключен физической кнопкой
• Неправильно выбран микрофон
• Микрофон не работает

Рекомендуем скажите что-нибудь для проверки.

Всё равно начать интервью?
```

Кандидат может:
- ❌ **Отменить** → проверить микрофон
- ✅ **Продолжить** → начать на свой риск

---

## 🔧 Технические детали

### **Новые состояния:**

```typescript
const [micTestedSuccessfully, setMicTestedSuccessfully] = useState(false);
```

Отслеживает, был ли хоть раз зафиксирован звук выше порога (5%).

### **Логика проверки:**

1. `ProductionWebcamComponent` непрерывно анализирует уровень микрофона
2. `onMicLevelChange(level)` передает уровень (0-100)
3. Если `level > 5` → индикатор зелёный, микрофон работает
4. Если `level < 1` при старте → предупреждение

### **Визуальная обратная связь:**

```tsx
<Box sx={{
  bgcolor: micLevel > 5 ? '#e8f5e9' : '#fff3e0',
  borderColor: micLevel > 5 ? '#4caf50' : '#ff9800'
}}>
  <Box style={{ width: `${Math.max(micLevel * 2, 2)}%` }} ... />
</Box>
```

## 🎯 Преимущества

1. **Кандидат видит** что микрофон работает ДО начала интервью
2. **Предупреждение** если физически отключен микрофон
3. **Меньше испорченных интервью** из-за проблем с звуком
4. **Лучший UX** - визуальная обратная связь

## 🧪 Тестирование

### **Сценарий 1: Микрофон работает**
1. Открыть страницу подготовки
2. Разрешить доступ к микрофону
3. ✅ Увидеть зелёный индикатор и анимацию уровня
4. ✅ Кнопка "Начать" активна

### **Сценарий 2: Микрофон отключен физически**
1. Открыть страницу подготовки
2. Разрешить доступ к микрофону
3. Нажать физическую кнопку "отключить микрофон"
4. ⚠️ Индикатор оранжевый, полоса не двигается
5. Нажать "Начать"
6. ⚠️ Получить предупреждение
7. Выбрать "Отменить" или "Продолжить"

### **Сценарий 3: Неправильный микрофон**
1. Открыть страницу подготовки
2. Система выбрала не тот микрофон
3. ⚠️ Индикатор не реагирует на голос
4. Кандидат видит проблему → может перезапустить или выбрать другой микрофон
