# Функционал системы автоматического рекрутинга

## Основные фишки (AI-интервью)

### 1. Автоматические интервью без участия интервьюера
- Полностью автоматизированный процесс проведения интервью
- ИИ ведёт диалог с кандидатом в реальном времени
- Адаптивные вопросы на основе предыдущих ответов

### 2. Анализ каждого ответа ИИ
- Оценка качества ответов
- Анализ полноты информации
- Проверка релевантности ответов
- Выявление противоречий в ответах

### 3. Анализ всего интервью ИИ
- Общая оценка кандидата
- Анализ коммуникативных навыков
- Оценка технических знаний
- Анализ soft skills

### 4. Ранжирование кандидатов по ответам
- Автоматическая сортировка по релевантности
- Сравнительный анализ кандидатов
- Рейтинговая система оценки
- Фильтрация по критериям

### 5. Дополнительные вопросы ИИ
- Система автоматически задаёт уточняющие вопросы
- Адаптивные вопросы на основе контекста
- Глубокий анализ ответов
- Выявление деталей и нюансов

### 6. Сравнение выбранных кандидатов
- Развёрнутый отчёт по кандидатам
- Анализ сильных и слабых сторон
- Сравнительная таблица характеристик
- Рекомендации по выбору

### 7. Генерация вопросов через ИИ
- Автоматическое создание вопросов на основе описания вакансии
- Адаптация вопросов под требования позиции
- Разнообразие типов вопросов
- Кастомизация сложности

### 8. Видео и аудио запись
- Запись интервью в реальном времени
- Возможность просмотра записей
- Анализ невербальной коммуникации
- Архивирование интервью
