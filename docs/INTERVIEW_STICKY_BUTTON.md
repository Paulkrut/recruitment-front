# Sticky кнопка на странице подготовки к интервью (Компактная версия)

## 📋 Проблема

На странице подготовки к интервью пользователю приходилось скроллить вниз, чтобы нажать кнопку "Начать интервью". После добавления sticky блока он занимал 2/5 экрана из-за юридических текстов.

---

## ✅ Решение: Минималистичная Sticky кнопка

```
┌─────────────────────────────┐
│ Описание вакансии           │
│ ✓ Согласия                  │
│ 📹 Превью                   │
│ 🎤 Проверка микрофона       │
│ 📄 Юридическая информация   │ ← переместили сюда
│    (ForgetMeAuto)           │
└─────────────────────────────┘
         ↓ скролл
╔═════════════════════════════╗ ← STICKY (минималистичная!)
║ ⏳ Проверяем микрофон...    ║
║ [Подготовка...]             ║
╚═════════════════════════════╝
```

**Высота sticky блока:** ~60-80px (было 400px → потом 200px → теперь 60-80px!)

---

## 🎯 Ключевые изменения

### V1 → V2 → V3:

**V1 (исходная):**
- Sticky блок: 350-400px ❌
- Содержимое: большие Alert + кнопка + ForgetMeAuto

**V2 (первая оптимизация):**
- Sticky блок: 150-200px ⚠️
- Содержимое: компактные статусы + кнопка + ForgetMeAuto

**V3 (текущая - оптимальная):** ✅
- Sticky блок: 60-80px ✅
- Содержимое: только компактный статус + кнопка
- ForgetMeAuto: перемещен в основной контент (выше sticky)

---

## 🔧 Реализация

### 1. ForgetMeAuto в основном контенте

```tsx
{/* Индикатор микрофона */}
{micReady && (
  <Box sx={{ ... }}>
    🎤 Микрофон работает
  </Box>
)}

{/* Юридическая информация - ПЕРЕМЕЩЕНА СЮДА */}
<Box sx={{ mt: 4, maxWidth: '600px', width: '100%' }}>
  <ForgetMeAuto candidateToken={token as string} />
</Box>

</Box> {/* закрывается основной контент */}

{/* Sticky блок */}
<Box sx={{ position: 'sticky', bottom: 0, ... }}>
```

### 2. Минималистичный sticky блок

```tsx
{/* Sticky Bottom Button - БЕЗ ForgetMeAuto */}
<Box sx={{
  position: 'sticky',
  bottom: 0,
  p: 2,
  bgcolor: 'background.paper',
  borderTop: '2px solid',
  zIndex: 100,
}}>
  {/* Только статус (если нужен) */}
  {!micReady && <Typography>⏳ Проверяем...</Typography>}
  
  {/* Только кнопка */}
  <Button fullWidth size="large">
    Начать интервью
  </Button>
  
  {/* ForgetMeAuto УДАЛЕН отсюда */}
</Box>
```

## 💡 Почему это лучше?

### Психология UX:
- **Основной контент**: юридические тексты можно прочитать при желании
- **Sticky блок**: только action (кнопка) - минимум отвлечения
- **Правило 80/20**: 80% пользователей не читают юридические тексты → убрали из sticky

### Технические причины:
- Меньше re-render для sticky блока
- Лучше производительность (меньше DOM в sticky)
- Проще поддержка
