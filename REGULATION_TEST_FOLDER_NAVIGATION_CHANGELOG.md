# 📋 Changelog: Навигация по папкам при создании теста регламентов

## [2025-11-04] - Добавлена навигация по папкам

### ✨ Новые возможности

#### 1. Отображение папок
- Папки отображаются на корневом уровне с большой иконкой 📁
- Показывается количество регламентов в каждой папке
- Визуальное отличие от регламентов

#### 2. Навигация по папкам
- Клик по папке открывает её содержимое
- Кнопка "⬅ Назад к корню" для выхода из папки
- Breadcrumbs показывает текущее местоположение: "/ Название папки"

#### 3. Выбор регламентов
- Можно выбирать регламенты из разных папок
- Выборы сохраняются при навигации
- Общий счётчик выбранных регламентов

#### 4. Пустые папки
- Сообщение "В этой папке пока нет регламентов"
- Можно вернуться назад и выбрать другую папку

### 🐛 Исправленные проблемы

#### 1. Длинный список регламентов
- **Проблема:** Слишком много регламентов в одном списке
- **Решение:** Структурирование по папкам

#### 2. Сложность поиска
- **Проблема:** Трудно найти нужный регламент
- **Решение:** Группировка по категориям (папкам)

#### 3. Отсутствие структуры
- **Проблема:** Все регламенты в одной куче
- **Решение:** Иерархическая навигация
