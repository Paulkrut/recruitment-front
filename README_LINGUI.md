## ⚡ Быстрый старт (5 команд)

```bash
# 1. Backup
git add . && git commit -m "backup before LinguiJS"

# 2. Проверить текущее состояние
npm run migrate:check

# 3. Запустить автомиграцию
npm run migrate:auto

# 4. Извлечь и скомпилировать
npm run i18n:extract-compile

# 5. Запустить и проверить
npm run dev
```

## 💡 Минимум, который нужно знать

### Для текста в JSX:

```tsx
import { Trans } from '@lingui/react';

<Typography><Trans>Мой текст</Trans></Typography>
```

### Для атрибутов:

```tsx
import { msg } from '@lingui/macro';
import { useLingui } from '@lingui/react';

const { _ } = useLingui();

<TextField placeholder={_(msg`Мой текст`)} />
```

### После изменений:

```bash
npm run i18n:extract-compile
```

## ⚠️ Важно помнить

1. **Всегда делайте backup** перед автомиграцией
2. **Проверяйте изменения** через git diff
3. **Тестируйте** после каждой миграции
4. **Коммитьте часто** - по файлам или группам
5. **Не пугайтесь** большого количества изменений

## 🆘 Проблемы?

1. Откройте **LINGUI_GUIDE.md** → раздел "Частые ошибки"
2. Откройте **LINGUI_AUTO_MIGRATION_HOWTO.md** → раздел "Устранение проблем"
3. Откатите: `git reset --hard HEAD`
